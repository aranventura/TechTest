Para ejecutar esta app solamente habria que crear un nuevo proyecto de react (tener Node.js y react) i volcar la inofrmación de los ficheros: main, index i SortMap.
Para ejecutarlo tendiramos que ir a la carpeta del proyecto i hacer npm start y se va a ejecutar en local en el puerto 3000.
